# LifeXP
### Level Up Your Real Life

LifeXP is a real-life gamification platform that helps users build better habits and prove real skills by converting daily activities into XP, levels, streaks, and achievements.

Unlike traditional social media, LifeXP focuses on **real growth**, not likes.
